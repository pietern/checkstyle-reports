3   /**
4    * Terminal stream operation that is aware of segments being lost.  This interface describes a single callback method
5    * to be invoked on the operation when a segment is lost and it is concurrently running some operation.
6    * @since 8.0
7    */
8   public interface SegmentAwareOperation {
9      /**
10      * This method will be invoked when the operation is known to be performing on a given set of segments
11      * and this node no longer owns 1 or many of them.  Returns whether the lost segment affected the results or
12      * not.  If stopIfLost is true then doneWithOperation will not be invoked as it would normally.
13      * @param allSegmentsLost argument to tell the operation that if this segment affects that it should not
14      *                   perform any more operations if possible as all segments have been lost.
15      * @return whether or not this operation was affected by the loss of segments
16      */
17     boolean lostSegment(boolean allSegmentsLost);
18  }
